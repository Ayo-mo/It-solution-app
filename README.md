# IT Solution App

## Project Overview

The IT Solution App is a Flask-based web application developed as a hands-on project to learn modern software development and DevOps practices. Rather than focusing only on building a website, this project follows the complete Software Development Life Cycle (SDLC), from development to deployment and monitoring.

The project is being built incrementally, with each phase introducing new tools, technologies, and engineering practices commonly used in professional software teams.

---

## Project Objectives

This project aims to:

* Build a dynamic web application using Flask.
* Learn Git and GitHub for version control.
* Develop and test the application in an Ubuntu Linux environment.
* Practice Python virtual environments for dependency management.
* Learn Docker containerization.
* Build CI/CD pipelines with GitHub Actions.
* Deploy the application to a production environment.
* Implement monitoring and logging tools.
* Understand how each stage fits into the DevOps lifecycle.

---

## Current Features

* Home page
* About page
* Services page
* Contact page
* Shared layout using Jinja2 templates (`base.html`)
* Static assets (CSS and images)
* Flask routing
* Form handling on the Contact page

## Running the Application

Clone the repository:

```bash
git clone https://github.com/Ayo-mo/It-solution-app.git
cd It-solution-app
```

Create a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python3 app.py
```

Then open the forwarded URL (typically http://127.0.0.1:5000) from VS Code.

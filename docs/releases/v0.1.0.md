# Release v0.1.0 – Dockerized Development Environment

**Release Date:** July 2026

## Summary

This release establishes the project's first reproducible development environment using Docker and Docker Compose.

## Features Delivered

* Dockerized the Flask application.
* Added a Dockerfile.
* Added a `.dockerignore` file.
* Configured Flask to listen on `0.0.0.0`.
* Added Docker Compose for local development.
* Verified browser access through published ports.
* Verified live development using bind mounts.

## Git Workflow

* Feature branch development.
* Small, logical commits.
* Pull Request review and merge.
* Merge commit preserved project history.

## Next Release

Version **v0.2.0** will introduce a database service, persistent storage, and environment variable management.
